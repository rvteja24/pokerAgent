Setting up on your local machine:

Step 1.) pip install pypokerengine.
Step 2.) git clone this project onto the local machine.
Step 3.) run the game_play file for the game to start.

Training: uncomment the code required for training and create a folder 'strategies/preflop' and run game_play.py
Testing/real-time gameplay: uncomment the code required for testing and run game_play.py

'Poker Agent Details' file contains a detailed explanation of all the techniques and their implementations used in building this agent.
